什么是MongoDB？
==============

MongoDB是一个面向文档的免费数据库，多用于数据采集和分散处理(Map/Reduce)，特别是在大数据处理方面比较擅长。

## 知识点

* 面向文档的数据库
* 面向关系的数据库

  + SQLServer
  + MySql
  + PostgreSql
* NoSql
  + MongoDB
  + Redis
